Руководство по запуску:
1) Сгенерировать скрипт базы данных в Microsoft SQL Server Management Studio 2012. Возможно в скрипте надо будет изменить путь к файлам, где будет храниться база данных.
2) Запустить проект в Microsoft Visual Studio 2012.
3) В файле WEB.config изменить строку подключения к базе данных, а именно имя сервера.В этом месте: data source=Sergey\SQLEXPRESS.
4) Запустить приложение.
